Spencers Redesign Project

This project focuses on improving the user experience, navigation, and visual design of the Spencers online platform.

Learn more about the issues identified and the solutions proposed to enhance Spencers' digital presence.

User Research

We conducted extensive user interviews, surveys, and usability testing to understand customer pain points with the current website.

  • Users find the navigation confusing.
  • Mobile users experience poor responsiveness.
  • The checkout process is too complicated.

Problem Statement

The Spencers website lacks modern design elements and a user-friendly navigation system, leading to poor user experience and lost sales opportunities.
